Опыт работы с технологиями

В ходе выполнения данного проекта автор работал с современным стеком технологий разработчика веб-приложений на Java:

  • Java 17+ — основной язык разработки;
  • Spring Boot 3 — создание backend-части, конфигурация приложения;
  • Spring MVC — построение контроллеров и маршрутов;
  • Spring Security — система аутентификации, авторизации и ролей;
  • Spring Data JPA / Hibernate — работа с базой данных;
  • PostgreSQL — хранение данных и проектирование структуры БД;
  • Thymeleaf — построение HTML-шаблонов;
  • Bootstrap 5 — вёрстка интерфейса;
  • Chart.js — визуализация статистики в разделе «Отчёты»;
  • MVC-архитектура — разделение модели, логики и представления.

В рамках проекта автор получил практический опыт в разработке CRUD-функционала, проектировании сущностей, построении связей между моделями, создании удобного пользовательского интерфейса и администраторской панели.
