Информационная система художественной школы

Портал для преподавателей и администрации: расписание занятий, журнал успеваемости, управление группами и отчётность.

Новости школы

Открытие учебного года

01.11.2025

Художественная школа открыла новый учебный год. Обновлены группы, расписание занятий и список дисциплин для начинающих и продолжающих.

Мастер-класс по живописи

10.11.2025

Преподаватель Иванов И.И. провёл открытый мастер-класс по масляной живописи для учеников базовых и продвинутых групп.

Обновление электронной системы

24.11.2025

Запущена новая версия информационной системы: добавлены журнал оценок, отчёты по успеваемости и удобная админ-панель.
